## Course Rules
- Fully typed
  - BANNED: `any`, `as`, `is`, `!`
  - ALWAYS: Write all type signatures (ie. `function strLen(s: string): number`)
- Adopt functional programming style 
  - No OOP, classes or attaching methods to object
- Golden Rule: Type First!!!
